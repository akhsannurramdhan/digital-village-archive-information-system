# SIADEK – Village Archive Management System

## Overview

SIADEK (Sistem Arsip Desa Elektronik) is a web-based archive management system developed to help village administrations digitize document management processes.

The system improves document organization, storage, retrieval, and administrative efficiency by replacing manual archive handling with a centralized digital platform.

## Problem Statement

Many village administrations still rely on manual document storage, making archives difficult to organize, search, and maintain.

This project was developed to:

* Improve archive accessibility
* Reduce document loss risks
* Accelerate administrative workflows
* Support digital transformation in village governance

## Features

* User Authentication
* Archive Management
* Document Upload & Storage
* Archive Search & Filtering
* Archive Categorization
* Administrative Dashboard
* Archive Monitoring
